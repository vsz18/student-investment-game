# Instructor Quick Start Guide

## Setup (Before Class)

1. **Access the Game**
   - Open browser and navigate to the game URL
   - Click "Instructor" button
   - Enter password: `zongzi`

2. **Upload Company Data**
   - Click "Choose File" button
   - Select your Excel file (.xlsx or .xls)
   - Wait for "Successfully loaded X companies" message
   - Verify companies appear in the list

3. **Keep This Tab Open**
   - ⚠️ Do NOT close or refresh this tab during class
   - Your session will be saved for 5 minutes if you accidentally disconnect

## During Class

### Phase 1: Setup (Before Students Join)
- Students see: "Waiting for instructor to start voting"
- You see: List of uploaded companies
- Action: Wait for students to connect

### Phase 2: Voting
1. Click **"Start Voting"** button
2. Students can now invest in companies
3. Monitor the **Active Students** panel:
   - Shows number of connected students
   - Shows each student's invested and remaining amounts
   - Shows total invested across all students
4. Watch the **Leaderboard** update in real-time

### Phase 3: Results
1. Click **"End Voting"** button
2. Final leaderboard is displayed
3. Click **"Export to Excel"** to download results
4. Save the Excel file immediately

## Monitoring Students

### Active Students Panel Shows:
- 👨‍🎓 Student name
- 💰 Amount invested (green)
- 💵 Amount remaining
- 📊 Total invested by all students

### What to Watch For:
- Students with $0 invested may need help
- Total invested should increase as students participate
- Remaining budget should decrease as students invest

## Common Actions

### If You Need to Refresh:
1. Refresh the page
2. Log in again with password
3. Your data will be restored (if within 5 minutes)
4. You'll see: "Session recovered! Your previous data has been restored."

### If Screen Freezes:
1. Wait 5-10 seconds
2. Don't close the tab
3. If still frozen, refresh and log back in

### If You Need to Reset:
1. Click **"Reset Game"** button
2. Confirm the reset
3. All student investments will be cleared
4. You'll need to re-upload the Excel file

## After Class

1. **Export Results**
   - Click "Export to Excel" in Results phase
   - Save the file with a meaningful name
   - File includes: Student name, Company, Investment, Comments

2. **Optional: Reset for Next Class**
   - Click "Reset Game" to clear all data
   - Upload new Excel file for next session

## Troubleshooting Quick Fixes

| Problem | Solution |
|---------|----------|
| Can't log in | Check password: `zongzi` |
| Excel upload failed | Check file format (.xlsx or .xls) |
| Students can't invest | Click "Start Voting" |
| Leaderboard not updating | Wait 2 seconds (auto-refresh) |
| Lost Excel data | Log back in within 5 minutes |
| Too many students | Max 69 students + 1 instructor |

## Tips for Success

✅ **DO:**
- Upload Excel before students join
- Keep browser tab open during class
- Monitor Active Students panel
- Export results immediately after voting
- Have backup Excel file ready

❌ **DON'T:**
- Close browser tab during class
- Refresh unless necessary
- Open multiple instructor windows
- Change phases too quickly
- Forget to export results
